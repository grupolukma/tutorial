Como cadastrar clientes

  1. Cadastro para clientes novos
  2. Clique no menu a esquerda com o nome Meus Prospects
  3. Clique no botão no canto superior direito Iniciar Cadastro
  4. Escolha Pessoa Física ou Pessoa Jurídica
  5. O botão Adicionar nova loja? serve para adicionar uma loja nova de um cliente que já existe, clicando nesse botão habilita uma caixa de pesquisa para selecionar o cliente
  6. Seguindo o cadastro normalmente insira os dados do cliente (Colocar número do CNPJ o restante ele pegara automatico), sempre preencher tudo com LETRA MAIÚSCULA
    Obs: Você só consiguirá fazer o cadastro de um cliente novo, se for um cliente que já existe na base de dados ligue na Lukma e peça para incluir o cliente na sua carteira
  7. Caso o cliente não possua Inscrição Estadual escreva "ISENTO" tudo maiúsculo
  8. Nos campos Código e Loja, em código colocar os 6 primeiros números do CNPJ, em loja colocar sempre 01.
  9. Após finalizar você poderá clicar no icone de 3 potinhos a esquerda do status é possível visualzar o cadastro do cliente e editar
    Obs: Se o cliente já possuir cadastro na Lukbox ligar na Lukma (17)2138-5050 e pedir para o pessoal do financeiro adicionar ele em sua carteira
