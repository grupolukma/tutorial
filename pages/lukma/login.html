Como fazer login no sistema Lukma

Este tutorial explica como acessar corretamente o sistema Lukma utilizando seu usuário e senha.


Passo a passo


  1. Abra o sistema Lukma.



  2. Se possuir mais de uma empresa, selecione a desejada.

    Para selecionar a empresa desejada fazer igual ao Tutorial do Trocar Empresa.



  3. Digite seu usuário e senha.



  4. Pressione o Enter uma vez para autenticar e um segundo Enter para entrar, ou clique no botão Entrar.

Vídeo explicativo

Em caso de erro de acesso, verifique seus dados ou entre em contato com o suporte.
