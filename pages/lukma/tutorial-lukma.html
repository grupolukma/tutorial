1️⃣ Login no sistema (login.html)

O que é / Para que serve

A tela de login é o ponto inicial de acesso ao sistema LUKMA. É por ela que o usuário entra no sistema e acessa o menu principal.

Como utilizar (passo a passo)

  1. Abra o navegador de internet
  2. Acesse o endereço do sistema LUKMA
  3. Informe seu usuário e senha
  4. Clique em Entrar
  5. Aguarde o carregamento do menu principal
Atenção: Caso os dados estejam incorretos, o sistema exibirá uma mensagem de erro.

2️⃣ Trocar de empresa

O que é / Para que serve

Permite alternar entre empresas cadastradas no sistema, garantindo que orçamentos, pedidos e produtos sejam vinculados à empresa correta.

Como utilizar (passo a passo)

  1. No menu principal, localize a opção Trocar de empresa
  2. Clique na opção
  3. Selecione a empresa desejada
  4. Aguarde a atualização da tela
Após a troca, todos os dados exibidos passam a ser da empresa selecionada.
